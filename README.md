# rphys-renderer

Declarative physics simulation videos from YAML scene definitions — built for short-form satisfying content.

## Crates

| Crate | Description |
|---|---|
| `rphys-scene` | YAML scene parser and data model |
| `rphys-physics` | Deterministic physics simulation (Rapier 2D) |
| `rphys-renderer` | CPU software renderer (tiny-skia) |
| `rphys-vfx` | Particle-based visual effects |
| `rphys-overlay` | Leaderboard and announcement text overlays |
| `rphys-race` | Race state tracking and scoring |
| `rphys-audio` | Event-driven audio engine |
| `rphys-export` | Video export pipeline (ffmpeg) |
| `rphys-preview` | Live preview window |
| `rphys-cli` | CLI entry point |
